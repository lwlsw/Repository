Ashley

Features:

    Reminder Options:

  • - swipe up to unlock
  • - no older notifications
  • - enter passcode

    • Applying Reminder:

  • - can be applied individually
  • - always apply everywhere
  • - always reset everywhere

    • Shake Gesture:

  • - on home- and lockscreen
  • - only homescreen
  • - only lockscreen

    • Timer:

  • - haptics when timer is over

    • Haptics:

  • - haptic animation
  • - custom strength

    • Talking:

  • - allow ashley to talk
  • - disable phrases individually

    • Other Options:

  • - Respring Button
  • - Reset Preferences Button

    • Preferences:

  • - beautiful design

Compatibility:

  • OS: iOS 13.0 - latest iOS 13
  • devices: all devices are supported
