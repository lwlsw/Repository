Dress

Features:

    Time And Date

  • - you can hide: time, date, lunar
  • - change alpha - opacity
  • - change alignment
  • - custom font / font size
  • - use rounded font
  • - use italic font
  • - use compact date format

    • FaceID Lock

  • - hide the lock
  • - change alpha - opacity
  • - alignment
  • - size

    • Status Bar

  • - hide the status bar
  • - change alpha - opacity

    • Homebar

  • - hide the homebar
  • - change alpha - opacity

    • Page Dots

  • - hide the page dots
  • - change alpha - opacity

    • Media Player

  • - hide the media player
  • - change alpha - opacity

    • CC Grabber

  • - hide the cc grabber
  • - change alpha - opacity

    • Notifications

  • - you can hide: no older notifications text, notification center text, notifications clear button
  • - change notification cells alpha - opacity
  • - change notification hint alpha - opacity
  • - change notifications header - opacity
  • - custom no older notifications text
  • - custom notification center text
  • - new scroll reveal animation
  • - hide dnd banner

    • Unlock Text

  • - hide the text
  • - custom text
  • - show last time unlocked
  • - configure last time unlocked

    • Quick Actions

  • - hide flashlight button
  • - hide camera button
  • - change flashlight button alpha - opacity
  • - change camera button alpha - opacity
  • - disable today swipe
  • - disable camera swipe
  • - custom x, y axis

    • Evanesco Mode

  • enable for: time and date, faceid lock, homebar, page dots, notification cells, no older notification text, notification header, unlock text, cc grabber, quick actions, complications, grupi, axon
  • - inactivity duration
  • - fade duration
  • - fade alpha - opacity

    • ColorFlow Coloring:

  • enable for: time and date, faceid lock, homebar, page dots, unlock text, quick actions

    • Custom Auto Lock Duration

  • - Default
  • - 10 Seconds
  • - 15 Seconds
  • - 20 Seconds
  • - 25 Seconds
  • - 30 Seconds

  • - Respring Button
  • - Reset Preferences Button

  • - beautiful design
