Rose

Features:

    Haptic Modes:

  • taptic engine modes: light, medium, heavy, soft, rigid
  • haptic engine modes: light, medium, strong
  • legacy engine modes: default, custom

    • Custom Legacy Mode Options:

  • - Custom Duration
  • - Custom Strength

    • Homescreen Haptics:

  • - 3D touch action/dismisses (iOS 12 only)
  • - opening folder
  • - closing folder
  • - tapping icon
  • - swiping to next/ previous page
  • - opening spotlight search

    • Lockscreen Haptics:

  • - Entering Passcode
  • - pressing quick actions button

    • Anywhere Haptics:

  • - killing app
  • - app switcher is used
  • - calling siri
  • - taking screenshot
  • - begin/end reachability
  • - using text selection
  • - entering power down view
  • - respringing
  • - typing
  • - entering homescreen
  • - alert appears
  • - alert disappears

    • Hardware Button Haptics:

  • - volume changed
  • - pressing only sleep button
  • - unmuting

    • Control Center Haptics:

  • - toggling connectivity toggle
  • - opening control center
  • - toggling cc module

    • Status Changes:

  • - unlocked/NC
  • - locked
  • - authenticated with touch/face id
  • - call connects/disconnects

    • - Other Hardware Actions Haptics:

  • - display wakes
  • - charger plugged in/out
  • - connecting bluetooth device in settings

    • System Elements Haptics:

  • - tapping uibutton
  • - tapping uibuttonbarbutton
  • - tapping uitabbarbutton
  • - tapping uiimageview
  • - tappig uiview
  • - tapping uiwindow

    • Applications Haptics:

  • currently supported apps: apollo, calculator, facebook, instagram, music, phone, safari, spotify, tiktok, twitter

    • Apollo Haptics:

  • - tapping the jump bar
  • - tapping the floating button
  • - tapping most elements
  • - tapping uibutton elements

    • Calculator Haptics:

  • - tapping a keypad button

    • Facebook Haptics:

  • - using the tabbar
  • - tapping quick access buttons
  • - tapping navigation bar buttons

    • Instagram Haptics:

  • - tapping like button
  • - double tapping to like
  • - tapping comment button
  • - tapping save button
  • - tapping send button

    • Music Haptics:

  • - tapping music control buttons
  • - tapping volume slider
  • - tapping more button
  • - tapping time slider
  • - tapping song cell
  • - tapping library cell
  • - tapping album cell
  • - tapping airplay button
  • - tapping lyrics, queue, play, skip buttons

    • Phone Haptics:

  • - tapping dialer number pad button
  • - tapping contacts cell
  • - tapping dialer delete button
  • - tapping dialer call button

    • Safari Haptics:

  • - tapping the url bar

    • Spotify Haptics:

  • - resuming/pausing song
  • - mini-player resuming/pausing
  • - going to previous song
  • - skipping song
  • - tapping repeat button
  • - tapping shuffle button
  • - tapping queue button
  • - using time slider
  • - tapping like button
  • - tapping devices button
  • - tapping song label
  • - tapping playlist buttons

    • TikTok Haptics:

  • - tapping like, comment, or share

    • Twitter Haptics:

  • - using tabbar
  • - tapping on a tweet
  • - tapping a dm chat
  • - tapping on a notification
  • - using the floating tweet button

    • Exceptions:

  • - when low power mode is on
  • - when dnd is on
  • - when ringer is silent

    • Additionals:

  • - haptic lock animation

    • Delay:

  • - you can choose to make rose have a delay

    • Other Options:

  • - Respring Button
  • - Reset Preferences Button
  • - Show Setup Again Button
  • - Show Quick Tips

    • Setup:

  • rose has a setup to help you getting started, it automatically enables the tweak and detects which device you're using to configure the engine settings for you

    • Preferences:

  • - beautiful design
  • - turn the blur view on or off

    • Rose Has Translations For Other Languages

    Every Haptics Option Can Have Its Own Strength

Compatibility:

  • OS: iOS 12 - latest iOS 13
  • devices: all devices are supported except iPods and iPads

Additional Information:

  • developer: Litten
  • current version: 6.0
  • last time updated: 05.05.2020
  • released: 09.28.2019
  • price: free
  • identifier: sh.litten.rose
  • dependencies: mobilesubstrate, preferenceloader, ws.hbang.common(>= 1.14), firmware(>= 12)
  • conflicts: none
